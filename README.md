Planejamento de Testes: Módulo de Autenticação
Sobre o Projeto
Este projeto apresenta o planejamento detalhado de testes funcionais e de segurança para um fluxo de login de uma aplicação web. O objetivo é demonstrar a aplicação de metodologias de Quality Assurance (QA) e a capacidade de análise de requisitos de software.

Competências Demonstradas:
Análise de Requisitos: Identificação de regras de negócio e critérios de aceite.

Escrita de Casos de Teste (CTs): Criação de cenários positivos e negativos com passos detalhados e resultados esperados.

Foco em Segurança: Validação de mascaramento de dados e prevenção de acessos indevidos.

UX/UI Testing: Verificação de alertas de erro e feedback visual para o usuário.
